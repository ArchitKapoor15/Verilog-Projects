# Verilog-Projects

This repository contains a growing collection of digital design projects implemented in Verilog HDL. These projects focus on fundamental hardware building blocks, RTL design principles, simulation, and verification using testbenches.
Each project is organized in its own folder containing:

- Verilog source files
- Testbench files
- Simulation waveforms
- RTL Schematic / Block Diagram
- Project documentation

## Projects

- **Synchronous FIFO**  
  A parameterizable synchronous FIFO module with full and empty flag generation.

*This collection will expand with new projects over time.*
